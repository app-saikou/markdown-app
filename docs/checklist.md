## リリース前チェックリスト（AdMob / 運用系）

### AdMob 設定

- [ ] AdMobアカウントでアプリを登録し、**本番App ID**を取得
- [ ] `app.json` の `expo.plugins` に本番App IDを設定
  ```json
  [
    "react-native-google-mobile-ads",
    { "androidAppId": "ca-app-pub-xxx~xxx", "iosAppId": "ca-app-pub-xxx~xxx" }
  ]
  ```
- [ ] `expo prebuild --clean` を実行してネイティブ再生成
- [ ] **バナー広告**の本番ユニットIDを取得 → `.env` に `EXPO_PUBLIC_ADMOB_BANNER_IOS` として設定
- [ ] **インタースティシャル広告**の本番ユニットIDを取得 → `.env` に `EXPO_PUBLIC_ADMOB_INTERSTITIAL_IOS` として設定
- [ ] `__DEV__` フラグでテストID/本番IDの切り替えを確認
- [ ] AdMobポリシー準拠確認（プライバシーポリシーURL・広告表示位置）

### ATT（App Tracking Transparency）

- [ ] ATTダイアログ文言の最終確認（`NSUserTrackingUsageDescription`）
- [ ] iOS 14以降でATTダイアログが正しく表示されることを実機確認

### リリースビルド確認

- [ ] `eas build --platform ios --profile production` でビルド通過確認
