# AIが雑なメモを磨き抜く — 思考を形にするメモアプリ

## コンセプト

このアプリの核心は、**あなたの断片的な思考をAIが「見せられる形」に変える**こと。

メモの時点では、考えていることを急いで書き殴るだけ。それで十分。後から、AIがそれを整理し、構造をつけ、他者に伝わる文章に昇華させてくれる——そういう仕組みです。

## できること

- **雑なメモ → 完成度の高いドキュメントへ**
  - 思考の断片を入力するだけで、論理的で読みやすいMarkdown形式に自動整理
- **複数のフォーマットで出力**
  - Markdown、その他の形式に対応
  - 個人の整理用から、他者・AIとの共有まで、用途に応じた出力が可能

- **AIに任せるから手間がかからない**
  - あなたは「思考を記録する」だけ
  - 「形にする」はAIの仕事
  - 著作権や知的財産は自分のもの

## こんな人向け

- アイデアが浮かぶたびに、急いでメモを取る人
- 後で見返すときに「これ何だっけ？」となる人
- 思考を整理したり、他人に説明する文書にしたりするのが面倒な人
- AIの力を使って、自分の思考の質を高めたい人

---

**つまり**：あなたの脳は「考える」に集中し、AIが「整える」を担当する。それが、このメモアプリの価値です。
