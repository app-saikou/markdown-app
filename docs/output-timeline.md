# AIが雑なメモを磨く——共有可能な高品質ドキュメントへ

## コアコンセプト

このアプリの本質は、**思いつきをプロダクトに変える**ことにあります。

ユーザーが走り書きした雑なメモを入力すると、AIが整理・構造化・表現の磨き上げを行い、そのまま他者やAIと共有できる品質に引き上げます。

## 主な機能

- **メモ入力** — 形式を気にしない、思いのままの記述が可能
- **AI処理** — 雑なテキストを論理的で読みやすいドキュメントに変換
- **複数フォーマット出力** — Markdown形式など、共有・転用しやすいファイル形式で書き出し可能

## 使う価値

| 場面               | 効果                                 |
| ------------------ | ------------------------------------ |
| **個人の思考整理** | ぼんやりした考えが言語化される       |
| **他者への説明**   | 聞き手が理解しやすい形で共有できる   |
| **AIとの協働**     | 次のAIタスクに適切な入力形式で渡せる |

## 一言で言えば

雑な入力 → **✨ AI編集 ✨** → 高品質アウトプット

クリエイティブな思考に集中でき、形式化の手間はAIに任せられるアプリです。
